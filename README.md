# Proyecto final - Agustín Bravo

Ecommerce desarrollado en React, Vite y Firebase.

## Descripción
Tienda online de indumentaria Hering (mujer, hombre, niños e íntima), con catálogo dinámico, carrito de compras persistente y checkout que genera una orden real en Firestore.

## Tecnologías
- React
- Vite
- React Router
- Firebase / Firestore
- React Toastify
- React Icons

## Funcionalidades
- Listado de productos por categoría
- Detalle de producto con selector de cantidad
- Carrito de compras global con Context API
- Checkout con validación y descuento de stock en Firestore
- Confirmación de compra con ID de orden
